transition
    전환시킨다
        요소에 추가할 css 스타일의 전환 효과 및 지속 시간을 설정
        css가 바뀌는 과정을 사용자가 눈으로 확인할 수 있도록 하는 속성
        ex) 상단 메뉴가 0 -> 300px로 전환되었을 때 높이가 0부터 300까지 커지는 과정을 사용자는 확인할 수 있다
        
        transition : all(height, width, opacity) 2s         ease                        2s
                     ---                         --         -----                       --
                     property                    duration   timing-function             delay
                                                            (전환 효과의 변경되는 속도 값)     (지연 시간)
                                                            빠르게 - 느리게 - 빠르게
        timing-function

            * ease (점점 느리게)
            * linear (평균 속도로)
            ease-in (점점 빠르게)
            ease-out (빠르게 느리게)
            * ease-in-out (빠르게 느리게 빠르게)
            step-start (시간초가 가기전에 한번에)
            step-end (시간초가 모두지난 후에 한번에)

animation
    애니메이션

    요소의 현재 스타일을 다른 스타일로 변화시킬건데
    반복, 시간의 경과에 따라 변화(10%, 30%, 50%) 컨트롤, 지연 시간 

    시간 경과에 따라 적용할 스타일은 무조건 초기 스타일이 기준 (30%라고 해서 이전 %인 10%에서 변화하는게 아닌, 0%)

    @keyframes 애니메이션명 {
        from 혹은 0% { ----------------> duration 값 비율 기준 
            시작 상태 
        }

        정수% (30%, 50%, 70%){
            중간 지점마다 적용할 CSS
        }

        to 혹은 100% {
            마지막 상태
        }

    }

    animation : 애니메이션명 duration(몇 초동안 변화) count(회, 무한대) direction(편도, 왕복)
